# 外壳尺寸与按键布局草图

## 1. 说明

这份草图文档不是最终机械图，而是把参考图片进一步落到结构层，给 3D 打印首版外壳和后续 PCB 布局提供一个稳定起点。所有尺寸都建议在打样前再按你实际购买的屏幕、开发板、编码器和键帽复核一遍。

## 2. 机身建议尺寸

第一版外壳建议从下面这组尺寸起步：

| 项目 | 建议值 |
|---|---:|
| 整体宽度 | `156 mm` |
| 整体深度 | `102 mm` |
| 前缘高度 | `18 mm` |
| 后缘高度 | `32 mm` |
| 顶板厚度 | `3 mm` |
| 外壳壁厚 | `2.4 mm` |
| 顶板倾角 | `7° ~ 9°` |

这组尺寸的目的，是在保持桌面小体积的前提下，把长条屏、两排主键和旋钮都容纳进去，同时保留一定走线与安装余量。

## 3. 屏幕区域建议

| 项目 | 建议值 |
|---|---:|
| 屏幕安装槽宽度 | `102 mm` |
| 屏幕安装槽高度 | `34 mm` |
| 屏幕可视开窗宽度 | `86 mm` |
| 屏幕可视开窗高度 | `20 mm` |
| 屏幕上边距 | `10 mm` |

如果你实际使用的 ST7789 模组边框更宽，优先以实物测量尺寸回调安装槽，不要反过来强行卡模组。

## 5. 按键宽度建议

为了更接近参考图的视觉比例，推荐采用下列宽度：

| 外观标签 | 建议键宽 |
|---|---:|
| `MIC` | `1.75u` |
| `ultrathink` | `2.25u` |
| `ESC` | `1.25u` |
| `claude` | `1.25u` |
| `Plan` | `1.25u` |
| `←` | `1.25u` |
| `Accept` | `2.75u` |

说明：

- `Accept` 需要明显更宽，作为视觉主按钮。
- `ultrathink` 需要比普通 1u 键更宽，强化“高频主动作”的位置感。
- `MIC` 建议比第二排普通键略宽，方便盲按。

## 6. 旋钮区域建议

| 项目 | 建议值 |
|---|---:|
| 旋钮中心距左边 | `18 mm` |
| 旋钮中心距前边 | `24 mm` |
| 编码器固定孔直径 | `7 mm` |
| 旋钮帽预留直径 | `22 mm` |
| 旋钮与最近按键净距 | `8 mm` |

旋钮建议略靠左下，并与第二排主键垂直视觉对齐。这样手感更接近参考图，不会和屏幕或第一排按键抢空间。

## 7. 首版 CAD 坐标建议

如果你准备开始画顶板，可以先用左前角作为 `(0, 0)` 原点，按下面这组参考中心点放置：

| 元件 | 中心坐标（x, y） |
|---|---|
| `MIC` | `(43, 57)` |
| `ultrathink` | `(87, 57)` |
| `ESC` | `(132, 57)` |
| `claude` | `(46, 84)` |
| `Plan` | `(76, 84)` |
| `BACK` | `(106, 84)` |
| `Accept` | `(137, 84)` |
| `KNOB` | `(18, 24)` |

这些坐标主要用于建立第一版草模，不建议直接当最终生产尺寸。

## 8. 结构建议

第一版打印时，优先保留以下结构特征：

- 顶板和底壳分体
- 屏幕区域独立压条或内侧卡台
- 主控区预留 USB 开口和线缆转角空间
- 旋钮区底部留足螺母与焊点高度
- 按键区尽量保留热插拔和换轴空间

## 9. 打样优先级

建议按这个顺序推进：

1. 先打一个只验证孔位和屏幕窗口的低精度草模
2. 再修正键帽间距与旋钮位置
3. 最后才做边角、装饰件、铭牌和配色细节

# 接线指南

## 1. 接线原则

本项目优先采用 `ESP32-S3 + ST7789 + 7 个独立按键 + 1 个旋钮` 的接线方式。样机阶段建议直接沿用参考固件常见的引脚分配，减少一开始就改代码的复杂度。

## 2. 推荐引脚映射

### 2.1 按键

| 逻辑键位 | GPIO | 对应外观 |
|---|---:|---|
| `MIC` | `GPIO0` | 麦克风图标键 |
| `CUSTOM` | `GPIO2` | `ultrathink` |
| `ESC` | `GPIO3` | `ESC` |
| `NEXT` | `GPIO4` | `Plan` |
| `BACKSPACE` | `GPIO5` | `←` |
| `SWITCH` | `GPIO6` | `claude` |
| `ACCEPT` | `GPIO7` | `Accept` |

连接方式：

- 每个按键一端接对应 GPIO
- 另一端接 `GND`
- 使用内部上拉

### 2.2 旋钮

| 功能 | GPIO |
|---|---:|
| 旋钮 A 相 | `GPIO16` |
| 旋钮 B 相 | `GPIO17` |
| 旋钮按压 | `GPIO18` |

### 2.3 屏幕

| 屏幕信号 | GPIO |
|---|---:|
| `MOSI` | `GPIO21` |
| `SCLK` | `GPIO47` |
| `CS` | `GPIO12` |
| `DC` | `GPIO13` |
| `RST` | `GPIO14` |
| `BL` | `GPIO11` |

### 2.4 可选麦克风

| I2S 信号 | GPIO |
|---|---:|
| `WS` | `GPIO41` |
| `BCLK` | `GPIO42` |
| `DIN` | `GPIO40` |

## 3. 接线顺序建议

1. 先接主控供电和屏幕。
2. 屏幕点亮后再接三个核心键：`ESC`、`ACCEPT`、`CUSTOM`。
3. 三个键工作稳定后，再补齐其余按键和旋钮。
4. 语音相关模块放到最后。

## 4. 调试注意事项

- `GPIO0` 和启动模式有关，调试时不要在上电瞬间长按对应按键。
- 屏幕刷新异常时，优先检查 `MOSI / SCLK / CS / DC / RST / BL`。
- 旋钮反向时，通常只需要交换 `A/B` 两路引脚。
- 如果按键持续误触，先排查接地、上拉和焊点，再看软件防抖。

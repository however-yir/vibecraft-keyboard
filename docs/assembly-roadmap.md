# 装配与迭代路线

## 1. 第一阶段：飞线样机

目标是尽快做出“能亮、能按、能连”的最小可运行版本。

建议完成以下检查：

- 屏幕点亮
- `ESC`、`ACCEPT`、`CUSTOM` 三个键可触发
- 旋钮方向可识别
- `vibekeys send` 能把文本发到屏幕

## 2. 第二阶段：布局定型

当样机链路打通后，再开始处理“像不像图片”的问题：

- 调整按键间距和行列关系
- 确认长条屏位置和边框厚度
- 放大 `Accept` 键帽，增强视觉中心
- 预留右上角装饰件或扩展位

## 3. 第三阶段：外壳与结构优化

建议通过 3D 打印逐步迭代：

- 第一版只验证装配
- 第二版优化握持、边角和屏幕开窗
- 第三版再考虑卡扣、走线和量产友好性

## 4. 第四阶段：软件强化

当硬件稳定后，可以继续增强：

- 模式页切换
- 不同模型名称显示
- 更多 Hook 事件上屏
- 自定义按键配置管理
- 状态灯或灯效提示

## 5. 当前优先级建议

如果你正在一步步推进，建议优先级保持如下顺序：

1. 接线
2. 点亮
3. BLE 发送文本
4. 默认键位可用
5. Hook 联动
6. 外观精修
